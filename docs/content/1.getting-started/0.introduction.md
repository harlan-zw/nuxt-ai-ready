AI agents like ChatGPT and Claude answer questions about your topics. When they do, you want your content cited.

Standards like [llms.txt](https://llmstxt.org/) and [MCP](https://modelcontextprotocol.io/) let agents query your site directly. Implementing them takes time.

Nuxt AI Ready converts pages to markdown, generates llms.txt at build time, and serves AI-friendly formats to bots automatically.

## Features

### llms.txt Generation

[llms.txt standard](https://llmstxt.org/) files generated at build time:

- `/llms.txt` — Site overview with page links and titles
- `/llms-full.txt` — Full markdown content for all pages
- [Content Signals](/docs/ai-ready/guides/content-signals) — AI usage permissions in `robots.txt`{lang="txt"}

### Markdown Serving

- **On-Demand Markdown** — Any route as `.md` (e.g., `/about` → `/about.md`)
- **Smart Bot Detection** — Markdown to AI crawlers, HTML to browsers
- **Cache Headers** — Stale-while-revalidate for performance

### MCP Integration

[Model Context Protocol](https://modelcontextprotocol.io/) server with:

- `list_pages` — Page metadata as JSON
- `search_pages_fuzzy` — Fuzzy search via Fuse.js
- `pages` resource — Page listing for agents
